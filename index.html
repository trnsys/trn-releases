Mac & Linux

curl -fsSL https://trnsys.github.io/trn-releases/install.sh | sh

Detects your platform, downloads the latest release, and installs to ~/.local/bin

To install elsewhere: curl -fsSL ... | TRN_INSTALL_DIR=~/bin sh

View install.sh

Manual installation

  1. Download the archive for your platform from the releases page
  2. Extract the binary (trn or trn.exe)
  3. Optionally add the install location to your PATH
