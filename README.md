# trn releases

Release binaries for [`trn`](https://github.com/trnsys/trn), the TRNSYS management tool.

This repo exists so `trn` can check for and download updates without authentication. Source code lives in the private [`trnsys/trn`](https://github.com/trnsys/trn) repo.

See [Releases](https://github.com/trnsys/trn-releases/releases) for downloads.
